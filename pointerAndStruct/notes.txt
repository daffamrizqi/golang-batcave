Pointer:
- Pointer adalah reference atau ALAMAT MEMORI
- Variable pointer berisi alamat memori suatu nilai
- ex : variabel int dengan value 4, maka yang dimaksud dengan pointer adalah
  alamat memori dimana value 4 disimpan, bukan nilai 4 nya sendiri

Penerapan Pointer:
- Variable pointer ditandai dengan adanya ASTERISK( * ) tepat sebelum
  penulisan tipe data ketika deklarasi
- Nilai default variable pointer adalah nill (kosong)
- Variable pointer tidak bisa menampung nilai yang bukan pointer, dan sebaliknya
- + Variable biasa bisa diambil nilai pointer nya dengan menambahkan tanda ampersand (&)
    tepat sebelum nama variable (metode referencing)
  + Variable Pointer juga bisa diambil value nya dengan menambahkan tanda asterisk(*)
    tepat sebelum nama variable (metode dereferencing)

Efek perubahan nilai Pointer:
- Ketika salah satu variable pointer di ubah nilainya, sedang ada variable lain yang
  memiliki referensi memori yang sama, maka nilai variable lain tsb juga akan berubah
